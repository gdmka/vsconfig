## Default config file path

### OSX

```
$HOME/Library/Application Support/Code/User
```
___

### Linux

```
$HOME/.config/Code/User/
```
___

### Win

```
%APPDATA%\Code\User\
```
___

## Reinstall extensions back

```
xargs -n1 code --install-extension < extensions
```
